# SeniorSidekick

An offline-first demo of a voice-and-text troubleshooting assistant for seniors.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Edit the demo tree

Change the placeholder instructions in `lib/decision-trees.ts`. The app needs no environment variables or external APIs in demo mode.

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or import this repository in the Vercel dashboard. The default build settings work as-is.
